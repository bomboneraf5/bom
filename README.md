# Reservas Canchas Sintéticas

Aplicación full-stack para gestionar reservas de 5 canchas sintéticas 24 horas.

## Scripts

- `npm install` → instala dependencias  
- `npm start` → arranca servidor en http://localhost:3000  

## Endpoints API

- `GET /api/reservations` → lista todos los slots  
- `POST /api/reservations` → crea nueva reserva  
- `PUT /api/reservations/:cancha/:time/status` → actualiza estado  

## Frontend

Abre http://localhost:3000 para ver la UI.
